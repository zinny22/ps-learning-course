# Week 1 - 수학 (JavaScript / Node.js 16+)

이 폴더는 Week 1 문제들의 정답 코드 모음입니다.

## 실행 방법 (BOJ)

```bash
node <파일명> < input.txt
```

예:

```bash
node boj-2438.js < input.txt
```
